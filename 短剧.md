# 《回声工牌》60s 悬疑宣传短剧脚本 v3

> 用途：竖屏悬疑宣传片 / 游戏引流短剧 / 漫剧预告
> 时长：60 秒
> 画幅：9:16
> 核心要求：开头使用“高楼远景坠落人影 + 记者韩沛现场播报 + 警戒线后紫色工牌”，全片不露正脸，不透露真相和结局。
> 视觉原则：用远景、人影、背影、手部、屏幕反光、工牌、警戒线、监控画面、空椅子代替人物正脸。
> 剪辑原则：新闻现场开场，工作台接管叙事，结尾停在“工牌即将拖入”前一帧。

---

## 宣传钩子

一名 24 岁男子，从南洄科技 C 座坠落。
记者韩沛赶到现场时，警戒线后只剩一张紫色工牌。
警方说排除刑事案件。
可那张工牌，第二天还在登录系统。

---

## 60 秒正式分镜

### 0-4s：高楼远景，人影坠落

- 画面：南洄科技园 C 座远景，傍晚偏冷色。楼体很高，人影只是远处一个黑点，从画面上方快速坠下。
- 镜头：超远景固定机位，不拍细节，不拍落地画面。人影坠出画面后，镜头不跟，留 0.3 秒空楼。
- 声音：没有尖叫，只有风声、电流底噪、远处一声急刹。
- 字幕一闪：
  - `6月6日 15时许`
  - `南洄科技园 C 座`
- 转场：急刹声切成摄像机开机的“滴”，画面跳到新闻现场。

### 4-10s：记者韩沛现场播报

- 画面：警戒线前，记者韩沛背对 C 座进行现场播报。镜头只拍她的半侧身、话筒、记者证，不给完整正脸。
- 背景：警戒线、模糊警灯、地面白色标线、远处围观人群全部虚焦。
- 镜头：中近景轻微手持，随后缓慢下移到地面。
- 韩沛播报：
  - “这里是南洄科技园 C 座。今天 15 时许，一名 24 岁男子从楼上坠落。警方初步排除刑事案件。”
- 画面重点：镜头下移到警戒线内侧，地面角落有一张紫色工牌，边缘沾灰，只能看见 `GL-0606` 的一角。
- 转场：工牌上的紫色反光铺满画面，切到工作台登录页的紫色背景。

### 10-16s：报道声接入登录页

- 画面：南洄登录页，紫色背景，图标“洄”。只拍键盘、手指、屏幕，不露脸。
- 镜头：顶拍键盘，手指输入四位口令，但画面切在按键节奏上，不正面停留完整密码。
- 声音：韩沛播报继续作为画外音，像从手机新闻里传来。
- 韩沛画外音：
  - “死者姓顾，曾为南洄智云科技有限公司员工，事发前登记为补交设备。”
- 屏幕文字：
  - `南洄市信息安全复核入口`
  - `正在验证旧系统口令`
- 转场：登录按钮变白，白光擦屏，切进工作台。

### 16-23s：工作台接管新闻叙事

- 画面：南洄工作台首页。消息、文档、AI 表格、AI 听记、待办、DING 依次亮红点。
- 镜头：
  - 从左侧导航甩镜到主界面。
  - 鼠标指针经过 `消息 99+`、`待办 10`、`DING 3`。
  - 每一个红点出现时短闪 2 帧。
- 屏幕文字：
  - `消息 99+`
  - `待办 10`
  - `DING 3`
  - `AI听记：摘要已生成`
- 旁白：
  - “新闻只说他回来补交设备。”
  - “但工作台记录了更多东西。”
- 转场：DING 图标一闪，切到门禁刷卡灯。

### 23-30s：紫色工牌成为主线

- 画面：紫色工牌特写，和新闻现场那张同款。桌面旁边是旧手机、冷咖啡、空工位。
- 镜头：
  - 微距扫过工牌边缘磨损。
  - 焦点从灰尘移到 `GL-0606`。
  - 姓名只露“顾□□□”，其余被污渍和反光遮住。
- 屏幕文字：
  - `GL-0606`
  - `Recovered Staff Credential`
- 低声台词：
  - “现场留下的，不只是遗物。”
- 声音：塑料卡片摩擦声，接一声门禁“滴”。
- 转场：工牌被拖入网页窗口，卡片边缘匹配切成监控画面边框。

### 30-37s：消息误导，第一层悬疑

- 画面：消息页。上属郑明的催方案消息出现，界面看起来很普通。
- 镜头：
  - 第一遍正常显示，画面稳定。
  - 第二遍同一构图回看，消息气泡边缘短暂出现暗红污染，立刻恢复。
  - 污染只闪 8-10 帧，不让观众读完整。
- 屏幕文字，正常版：
  - `顾路，C座异常路径复核方案今晚给我。`
  - `明天交接别再拖，设备和方案一起补齐。`
- 旁白：
  - “所有人都会先怀疑这条消息。”
- 声音：键盘声逐渐变成急促敲门声。
- 转场：敲门声最后一下切到文档修订光标。

### 37-45s：证据连环转场

- 画面：日历、待办、门禁、AI 听记、文档修订、旧版访客记录高速交叉。
- 镜头与转场：
  - 日历 `14:30` 被红框圈住，红框横向拉成待办进度条。
  - 待办 `已完成` 印章砸下，印章声切成门禁灯闪烁。
  - 门禁日志向上滚动，滚动条匹配切到 AI 听记波形。
  - 波形最后几秒断掉，断点变成文档里的删除线。
  - 删除线擦过屏幕，露出“补交设备”四个字。
- 屏幕短字：
  - `14:30`
  - `旧版路线`
  - `摘要已修订`
  - `回执不存在`
- 旁白：
  - “没有人说让他上去。”
  - “但每一条记录，都只剩一个方向。”
- 转场：最后一次门禁滴声没有响，只剩空调低频，切回新闻现场。

### 45-51s：韩沛的报道被打断

- 画面：韩沛站在警戒线外继续播报，依旧不露完整正脸。她低头看手机，屏幕光映在话筒金属边缘。
- 镜头：从话筒上的“韩沛”记者签，推到她手机里的南洄工作台通知。
- 手机通知：
  - `旧版访客记录已恢复`
  - `请刷入工牌`
- 韩沛停顿半秒，播报声断开。
- 韩沛低声：
  - “等一下……这条记录不是通报里的。”
- 转场：手机屏幕白光闪一下，切到旧窗口弹出。

### 51-56s：旧记录弹窗，终极疑问

- 画面：南洄工作台弹出旧版访客记录窗口。
- 镜头：
  - 快速推近，但弹窗遮住关键字段。
  - 鼠标移到“刷入工牌”，还没点。
  - 屏幕反光中出现一个站在门口的背影，下一帧消失。
- 屏幕文字：
  - `该记录不属于当前账号`
  - `GL-0606 凭证待刷入`
- 声音：心跳一声，随后全片静音 0.5 秒。
- 转场：静音中硬切黑屏。

### 56-60s：停在真相前

- 画面：黑屏里只有拖拽窗口轮廓。`GL-0606.html` 悬在窗口上方，鼠标松手前定格。
- 屏幕文字逐字出现：
  - `别急着相信第一条记录。`
  - `把工牌拖进去。`
- 片名：
  - `《回声工牌》`
  - `南洄工作台解密游戏`
- 声音：黑屏后补一声很轻的 DING，像系统还在运行。

## 无露脸拍摄规则

- 坠落开场只用远景人影，不拍落地细节，不拍面部。
- 韩沛可以出镜，但不露完整正脸，只拍侧身、背影、话筒、记者证、手和手机。
- 现场围观人员全部虚焦或只拍脚步、胸牌局部、警戒线外背影。
- 顾路本人不出镜，只通过紫色工牌、空工位、旧手机、未接来电、冷咖啡和系统记录存在。
- 家属或工作人员如需出现，只拍手、肩线、电话、桌面、玻璃反光。

---

## 高悬疑剪辑手法

- **新闻开机转场**：急刹声切成摄像机开机“滴”，从坠落远景转新闻现场。
- **颜色匹配**：紫色工牌反光匹配南洄登录页紫色背景。
- **J-cut 声音先行**：韩沛播报先于画面进入登录页，形成“新闻在追着系统走”的感觉。
- **匹配剪辑**：工牌边缘匹配监控边框；红框匹配待办进度条；印章声匹配门禁灯。
- **遮挡转场**：警戒线横穿画面、网页选中文字、手机亮屏白光都可做遮挡切镜。
- **同构误导**：同一条催方案消息出现两次，第二次短暂污染，让观众误判但不确认。
- **静音断崖**：旧记录弹窗出现后静音，片尾 DING 才回来。

## 声音设计

- 主声音：风声、急刹、摄像机开机滴、韩沛现场播报、DING、门禁滴声、键盘、手机震动、办公室空调。
- 坠落镜头不使用尖叫和冲击声，保持克制，让远景更冷。
- 韩沛播报前半段保持新闻专业腔，45 秒后第一次出现停顿和低声，制造裂缝。
- 证据闪切段音乐加速，但人声保持冷静，形成反差。
- 片尾 DING 要非常轻，像后台系统仍在运行。
